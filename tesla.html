Analisis Data Saham Tesla (2010)

1. Gambaran Dataset

Dataset ini berisi data historis saham Tesla dari 29 Juni 2010 hingga 23 Juli 2010 dengan kolom:

  • Date: Tanggal pencatatan
  • Open: Harga pembukaan
  • High: Harga tertinggi harian
  • Low: Harga terendah harian
  • Close: Harga penutupan
  • Volume: Volume perdagangan
import pandas as pd

# Load data
df = pd.read_csv('Tesla_stock_data.csv')
print(df.head())

2. Statistik Deskriptif

Analisis statistik dasar untuk memahami distribusi data:

# Statistik deskriptif
stats = df.describe()
print(stats)

Hasil statistik deskriptif menunjukkan rata-rata harga penutupan $1.31 dengan volatilitas (std dev) $0.16

3. Visualisasi Penting

3.1 Pergerakan Harga Saham

import matplotlib.pyplot as plt

plt.figure(figsize=(12,6))
plt.plot(df['Date'], df['Close'], label='Close Price', marker='o')
plt.title('Perkembangan Harga Saham Tesla (Juni-Juli 2010)')
plt.xlabel('Tanggal')
plt.ylabel('Harga ($)')
plt.xticks(rotation=45)
plt.grid(True)
plt.legend()
plt.show()

Terlihat penurunan harga signifikan di awal Juli sebelum stabil di pertengahan bulan

3.2 Korelasi Volume vs Harga

import seaborn as sns

plt.figure(figsize=(10,6))
sns.scatterplot(x='Volume', y='Close', data=df, hue='Date')
plt.title('Hubungan Volume Perdagangan vs Harga Penutupan')
plt.show()

4. Analisis Regresi Linier

Menganalisis pengaruh volume perdagangan terhadap harga penutupan:

import statsmodels.api as sm

# Regresi linier
X = df['Volume']
y = df['Close']
X = sm.add_constant(X)  # Tambah konstanta
model = sm.OLS(y, X).fit()
print(model.summary())

Hasil regresi menunjukkan R-squared 0.227 yang berarti hanya 22.7% variasi harga bisa dijelaskan oleh volume

5. Kesimpulan dan Rekomendasi

Temuan Utama:

  • Harga saham Tesla mengalami penurunan signifikan pada awal Juli 2010
  • Volume perdagangan memiliki pengaruh terbatas terhadap harga (R-squared rendah)
  • Terdapat fluktuasi harian yang cukup tinggi (standar deviasi $0.16)

Rekomendasi:

  • Perlu analisis faktor eksternal yang memengaruhi penurunan harga awal Juli
  • Model prediksi bisa dikembangkan dengan menambahkan variabel lain selain volume
  • Analisis time series (ARIMA) mungkin lebih cocok untuk data saham
